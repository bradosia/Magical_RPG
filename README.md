# Dependencies

## Debian
- g++

## Windows
- cygwin

# Build Instructions

## Debian
```sh
cd Ravioli_Racing\Release
g++ ../src/Ravioli_Racing.cpp ../src/POSIX_HTTP.cpp -o Ravioli_Racing
```

## Windows
```sh
cd Ravioli_Racing\Release
make
```

# Run Server

## Debian
```sh
cd Ravioli_Racing\Release
./Ravioli_Racing
```

## Windows
```sh
cd Ravioli_Racing\Release
Ravioli_Racing
```
